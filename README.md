# Note Ninja  

Note Ninja is a smart note-taking app built with **Next.js**, styled with **Tailwind CSS**, and powered by **AI (OpenAI)**. It lets you create, organize, and enhance notes with AI suggestions — perfect for students, developers, and lifelong learners.  

---

## Features  
- Create, edit, and delete notes easily  
- Search notes with AI-powered assistance  
- AI integration using **OpenAI API**  
- Modern UI styled with **Tailwind CSS** (custom light & dark purple themes)  
- Authentication & user management  
- PostgreSQL database with **Prisma ORM**  
- Fast & optimized with **Next.js**  
- Deployable on **Vercel**  

---

## Tech Stack  
- [Next.js](https://nextjs.org/) – React framework for the web  
- [React](https://react.dev/) – UI library  
- [Tailwind CSS](https://tailwindcss.com/) – Styling & theming  
- [Prisma](https://www.prisma.io/) – Database ORM  
- [PostgreSQL](https://www.postgresql.org/) – Database  
- [OpenAI API](https://platform.openai.com/) – AI assistance  
- [pnpm](https://pnpm.io/) – Package manager  
- [Vercel](https://vercel.com/) – Deployment

## Getting Started  

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).  

### Run locally  

1. Clone the repo  
   ```bash
   git clone https://github.com/your-username/note-ninja.git
   cd note-ninja
   ```

2. Install dependencies
  ```bash
  pnpm install
  # or
  npm install
  # or
  yarn install
```

2. Run the dev server
  ```bash
  pnpm dev
  # or
  npm run dev
  # or
  yarn dev
  # or
  bun dev
```
4. Open http://localhost:3000 with your browser.
